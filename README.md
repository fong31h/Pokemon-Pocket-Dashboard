# Pokemon Card Game Data Presentation

My third project was a passion project. I webscraped, cleaned, and analyzed tournament data from the mobile game Pokémon Trading Card Game Pocket. Then I built a dashboard in Python Shiny to present
the competitive history of the first year of this game's life. Please see my dashboard [here](https://019cf8ca-d476-452b-cd6c-e8061766ff0e.share.connect.posit.cloud)

Why does this project matter? First of all, this project shows my ability to tell a story with data. With this dashboard I tell a complete story of the first year of Pokémon TCG Pocket. 
Tournament results, changes in the metagame overtime, nuances in deckbuilding, and overall power rankings are all represented. Many people responded to my storytelling ability when I 
shared this dashboard on social media, saying it was comprehensive, informative, and interesting and fun to look at. I can use these skills to clearly
tell business stories, pinpoint critical information, and handle difficult data questions for your business.

Secondly, this project shows my ability to adapt to using new tools. While I am well-versed in the industry standards of SQL, Python, Excel, and Tableau, I learned a new tool, Shiny, which
I had never used before to tell this story. I can quickly adapt to new tools such as AI or other tools.

Anyways, I know this isn't the most business relevant project, but I worked hard on it and it seemed a shame to not include it.
